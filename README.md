# Multilayer Multistate Reaction Coordinate (ML-MRC) Model
Data repository for ML-MRC model

`plot.ipynb` is Jupyter Notebook for result plotting, the figures are saved to `Figures`.

Results for different testing cases are saved at:
- `equivalence`, equivalence test between ML-MRC with various layer number $L$ .
- `truncation`, truncation test between truncated ML-MRC with various truncation number $N_t$ .
- `RC`, for reaction coordinate plotting.
- `LAS`, linear absorption spectra.
